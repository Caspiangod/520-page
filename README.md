# 520 浪漫表白页

一个粉色梦幻风格的 520 表白页面，包含粒子爱心、樱花飘落、流星雨、烟花等动效，支持背景音乐与移动端适配。

## 预览效果

- 多层粉色渐变天空 + 散景光斑
- 200 颗会呼吸闪烁的星星 + 粉色拖尾流星
- 700 个粒子构成的脉动爱心
- 60 朵 Canvas 绘制的五瓣樱花飘落
- 鼠标/触摸轨迹粉色粒子拖尾 + 飘字
- 旋转唱片背景音乐控制
- 周期自动烟花

## 使用方法

1. 准备一份背景音乐 mp3 文件
2. 重命名为 `song.mp3` 放到本目录
3. 启动本地服务器：

   ```bash
   python3 -m http.server 8520
   ```

4. 浏览器访问 <http://localhost:8520>

## 技术栈

- 纯 HTML + CSS + Canvas
- 零依赖
- 已做移动端适配（视口、触摸、性能降配、DPR 高清）
